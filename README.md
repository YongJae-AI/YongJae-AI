**기간:** 2023.03 – 2023.09

- **관성반경:** 0.35B ~ 0.46B  

### 사용 장비  
- OptiTrack Motion Capture  
- Intel RealSense L515 / D435  
- MATLAB 기반 필터링, 스펙트럼 분석
